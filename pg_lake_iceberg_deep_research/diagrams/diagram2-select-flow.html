SELECT 查询端到端流程

从用户 SQL 到 DuckDB 向量化扫描再到流式返回——延迟文件绑定是核心设计

延迟文件绑定

  • • 规划时只放占位符，不绑定文件
  • • executor start 时才解析具体文件
  • • 保证 prepared statement 正确性

文件裁剪

  • • Iceberg 分区值 + 列 min/max 统计
  • • PG predicate proof 保留可能文件
  • • 裁剪在 DuckDB 扫描前完成

Delete Anti-Join

  • • 读数据文件时发射 filename + row_number
  • • 与 delete 文件做 NOT IN anti-join
  • • 向量化过滤，对用户隐藏
