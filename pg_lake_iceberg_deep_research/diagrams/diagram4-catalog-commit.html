Catalog 角色与提交流程

Iceberg catalog 是权威的名→表状态服务；pg_lake 实现 5 类 catalog 能力，提交通过事务钩子协调

内部 Catalog

  • • PG 目录表 = Iceberg catalog
  • • PG 行锁 = CAS 语义
  • • iceberg_tables 视图 JDBC 兼容

REST Catalog

  • • 可读写外部 catalog (Polaris)
  • • COMMIT 后发送 REST 请求
  • • 单事务限单一 REST catalog

关键限制

  • • Prepared tx 含 Iceberg 变更 → 拒绝
  • • REST post-commit 失败 → warning
  • • catalog.json 无冲突处理
