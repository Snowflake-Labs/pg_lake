# Claude Skills for pg_lake Development

This directory contains Claude skills that provide specialized knowledge for developing PostgreSQL extensions.

## Available Skills

### postgres-extension-dev
General PostgreSQL extension development guidance covering:
- Extension architecture and structure
- C programming patterns for Postgres
- PGXS build system
- Memory management and error handling
- Testing strategies

**Use when:** Working on any PostgreSQL extension, implementing new C functions, or learning extension development patterns.

## How Skills Work

Claude skills are automatically available in your Claude Code conversations. Simply mention topics related to the skills (e.g., "How do I add a new Iceberg type?" or "What's the pattern for registering a PostgreSQL function?") and Claude will use the relevant skill context to provide accurate, project-specific guidance.

## Maintaining Skills

As the project evolves, update these skills to reflect:
- New architectural patterns
- Additional testing approaches
- Changed conventions or best practices
- New components or features

Keep skills focused and practical - they should help developers quickly understand and follow project conventions.
